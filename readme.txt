Static image map is an easy to use plugin to insert a static map on your site.

== Description ==

Static image map is an easy and simple plugin that will download a Google Map image to your web site.
This image map will be downloaded again only if settings are changed. On the frond end no Google Map Javascript ressource
are use. 

== Privacy notices ==

This plugin does not:

* track users by stealth;
* write any user personal data to the database;
* send any data to external servers;
* use cookies.

== Translations ==

- English - default

- French

== Installation ==

1. Upload the entire 'static-image-map' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the **Plugins** screen (**Plugins > Installed Plugins**).

You will find **Static image map**  in your **Settings**  Screen (**Settings > Static image map** ) .
